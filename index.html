HTML

  • The head element contains metadata information about the webpage.
  • Metadata aids in accessibility, search engines, and performance.
  • The body element represents the visible content shown to the user.
  • The body element is an example of a semantic HTML element - the name of the element describes its content and the role it plays on a webpage.
  • HTML elements are represented by tags.
  • HTML elements can have attributes that provide additional information about the element.
  • HTML elements can be nested inside other elements.
  • The title tag within the head element defines the webpage title which is crucial for SEO.
  • The style tag within the head element defines CSS styles.
  • The base tag within the head element defines the base URL.
  • The link tag within the head element contains external resources.
  • The script tag within the head element contains JavaScript code.
  • The meta tag within the head element defines various metadata like character set, description, keywords, author, and viewport.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used to style the visual presentation of a webpage.
  • CSS can be applied to HTML elements using selectors.
  • CSS properties define the style of an element.
  • CSS values define the appearance of an element.
  • CSS rules are made up of a selector and a declaration block.
  • CSS declarations are made up of a property and a value.
  • CSS can be applied to HTML elements using inline, internal, and external styles.
  • CSS external styles are the most common and preferred method of applying CSS. It allows for easier design changes and adheres to the concept of the Separation of Concerns.
  • CSS can be applied to HTML elements using classes and IDs.
  • CSS can be applied to HTML elements using the box model.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The * selector is a wildcard selector which applies the style properties to all elements on a page.

Git

  • Git is a version control system that tracks changes in files and directories.
  • Git is used to collaborate with others on projects.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add .: stages all changes in the working directory
  • git commit -m "message": commits the staged changes
  • git push origin branch-name: pushes the changes to the remote repository
  • git pull origin branch-name: pulls the changes from the remote repository

JavaScript

  • JavaScript is a programming language that is used to make web pages interactive.
  • A variable is a named container that allows us to store data.
  • Variables can be declared using the var, let, or const keywords.
  • Variable types can be changed after they are declared, simply by assigning a new value to them of the new type. This concept is known as dynamic variables.
  • Conditional statements in JavaScript, such as if statements, evaluate if the statement is truthy or if the value of the statement itself is truthy. If it is, the code contained within the statement will be executed. If not, it will be ignored.
  • A value is considered falsy if it is one of the following values: 0, -0, "", null, undefined, NaN, or a Boolean data type with the value of false. All other values are considered truthy.
  • Loops in JavaScript, such as for loops, allow us to repeat a block of code multiple times.
  • JavaScript can be used to handle events like clicks, scrolls, and hovers.
  • JavaScript can be used to fetch data from APIs (Application Programming Interfaces).
  • JavaScript can be used to store data in local storage.
  • JavaScript can be used to create animations and transitions.
